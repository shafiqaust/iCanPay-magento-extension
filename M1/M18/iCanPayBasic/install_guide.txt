1. Create 'local' folder if it is not available in 'app/code/' directory. [Ex: app/code/local]
2. Place 'Mageapps' folder to app/code/loal/  [Example: {MagentoDir}/app/code/local/{Mageapps}]
3. If there is already 'Mageapps' folder then, apply 'Icanpay' folder into 'Maageapps' folder.[Ex: {MagentoDir}/app/code/local/Mageapps/{Icanpay}/], Otherwise, apply 'Mageapps' folder directly. [Ex: {MagentoDir}/app/code/local/{Mageapps}/]

4. app/etc/modules/Mageapps_Icanpay.xml    to    {MagentoDir}/app/etc/modules/{Mageapps_Icanpay.xml}

5. app/design/adminhtml/default/default/layout/icanpay.xml  to {MagentoDir}/app/design/adminhtml/default/default/layout/{icanpay.xml}
6. app/design/adminhtml/default/default/template/icanpay  to   {MagentoDir}/app/design/adminhtml/default/default/template/{icanpay}/ [Note: icanpay is a directory]
7. app/design/frontend/base/default/layout/icanpay.xml to   {MagentoDir}/app/design/frontend/base/default/layout/{icanpay.xml}
8. app/design/frontend/base/default/template/icanpay to  {MagentoDir}/app/design/frontend/base/default/template/{icanpay} [Note: icanpay is a directory]
9. Remove cache and session from command line or admin panel.
10. Go to admin panel and navigate to "System->Configuration->Payment Methods"
11. Put required api credentionls to the iCanPay section
12. This payment option will show in checkout page 
13. Bingo!
